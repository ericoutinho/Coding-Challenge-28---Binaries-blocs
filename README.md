# Coding Challenge #28 - Binaries blocs

From [@nextgencoder](https://www.instagram.com/p/B1Rp5vsA3Jn/)

Escreva a program which will take listaof numbers containin only 1s and 0s and counts number os blocks in which 1 is repeated more than once.

```
[1,0,0,1,1,1,0,1] => 1
[1,0,0,1,1,0,1,1] => 2
[1,1,0,1,1,1,0,1] => 2
```
